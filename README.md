# 💕 末末记账本 v2.0

一个温暖可爱的记账应用，集邮箱登陆、个人信息、消费记录、数据统计为一体。就像写日记一样简单，用粉色系的手账风设计，让记账变成一件温暖而有趣的事情。

## ✨ 功能特性

### 🔐 **用户认证系统**
- ✅ 邮箱注册和登陆
- ✅ 个人信息页面
- ✅ 可爱的动态头像生成
- ✅ 个性化座右铭和个人简介

### 📝 **日记式记账**
- ✅ 像写日记一样简单直观地记录每一笔消费
- ✅ 支持支出和收入两种类型
- ✅ 8个可爱的消费分类（餐饮🍜、交通🚗、购物🛍️、娱乐🎬 等）
- ✅ 日期、备注、金额完整记录
- ✅ 历史记录查看和删除功能

### 📊 **数据统计与报告**  
- ✅ **周报告**：周度支出、收入统计
- ✅ **月报告**：月度详细财务总结
- ✅ **智能建议**：根据消费数据自动生成理财建议
- ✅ 支出趋势图表、分类占比饼图
- ✅ 日均消费评估和优化建议

### 🎨 **设计特点**
- ✅ 温暖粉色系手账风格（奶油色→粉色→暖粉色渐变）
- ✅ 圆润可爱的卡片设计
- ✅ 平滑的交互动画效果
- ✅ 响应式布局，适配各种屏幕

### 💾 **数据管理**
- ✅ 本地存储（LocalStorage），隐私有保障
- ✅ 自动保存，无需担心数据丢失
- ✅ 用户数据与交易数据独立存储

```bash

```

### 开发调试

然后在浏览器中打开 `http://localhost:5173`

### 生产构建
```bash
npm run build
```

构建后的文件在 `dist` 目录中。

## 📊 功能使用

### 1. 注册和登陆
- 打开应用后，进入登陆/注册页面
- 可以选择"登陆"（使用已有邮箱）或"注册"（新建账户）
- 填写邮箱和用户名（注册时需要）
- 点击登陆或注册进入应用

### 2. 设置个人信息
- 点击右上角头像进入个人信息页面
- 编辑用户名和座右铭
- 保存更改

### 3. 记录消费
- 在首页点击 "新增记录" 按钮
- 选择消费类型（支出/收入）
- 选择日期和分类
- 输入金额和可选备注
- 点击保存

### 4. 查看周报告
- 切换到"周报告"标签页
- 查看本周的消费总结
- 了解各类别的消费占比
- 阅读智能建议

### 3. 查看月报告
- 切换到"月报告"标签页
- 查看本月的详细统计
- 分析消费趋势
